## Descrição

O **Jokengame** é um jogo clássico de Pedra, Papel e Tesoura (também conhecido como Jokenpô) desenvolvido inteiramente em HTML, CSS e JavaScript. O jogo permite que um jogador humano compita contra o computador em uma batalha emocionante de escolhas estratégicas. Insira os nomes dos jogadores, faça sua escolha e veja quem sai vencedor!

Este projeto é ideal para iniciantes em desenvolvimento web, demonstrando conceitos fundamentais de manipulação do DOM, eventos e lógica de jogo simples.

## Funcionalidades

- Jogo para dois jogadores: Um jogador humano contra o computador
- Inserção de nomes: Personalize o nome do jogador humano e do computador
- Escolhas visuais: Botões intuitivos com emojis para Pedra (✊), Papel (✋) e Tesoura (✌️)
- Resultado em tempo real: Exibe o resultado de cada rodada e a escolha do computador
- Placar persistente: Mantém a pontuação ao longo das rodadas
- Reinício fácil: Botão para voltar à tela inicial e redefinir nomes e placar
- Interface responsiva: Design adaptável para diferentes tamanhos de tela
